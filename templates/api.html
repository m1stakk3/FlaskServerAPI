GET

ip:8999/api/logs
Return list of all logs in JSON. Result example:
{
"id": int: {
"timestamp": string,
"message": string}
}

POST

ip:8999/api/logs
Input log row in JSON. Body example:
{"message": string}

DELETE

ip:8999/api/logs
Delete logs and return empty JSON
